Git sama saja dengan Version Control Sistem

Istilah - istilah pada github
- repo = folder project
- commit = rekaman ubah pada folder atau file 
- hash = penanda unik pada sebuah commit
- chechout = berpindah kesebuah commit
- branch = cabang bebas
- merge = menggabungkan branch
- remote = seumber yang memiliki repo
- clone = mengambil repo dari remote
- push = mengirimkan commit ke repo
- pull = mengambil commit ke repo

Istilah Git commit (local)
- $ git init
- $ git add <file>
- $ git add .
- $ git status
- $ git commit
- $ git config
- $ git branch
- $ git help
- $ git rm --cached <file>
- $ git config --global user.name "your name"\
- $ git config --global user.email "your email"
- $ git log
- $ git log -3 (angka bebas)
- $ git log --<file>
- $ git chechkout (5 digit hash)
